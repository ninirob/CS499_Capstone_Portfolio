Milestone One

Code Review

A walkthrough of all three artifacts as they stood before enhancement: a Java pet-boarding application, a CIFAR-10 image classification CNN, and a relational RMA database. For each, I trace the existing structure, identify concrete issues, and outline the plan for enhancement.

What the review covers

Software Design

Pet BAG Java Application

A constructor mismatch and a missing method, the result of two files from separate assignments never being integrated.

Algorithms

CIFAR-10 CNN Notebook

A silent Conv2D argument bug affecting stride behavior, and a stale evaluation variable masking the augmented model's real results.

Databases

Quantigration RMA Database

Literal-value queries with no parameterization, hardcoded file paths, and no least-privilege access control.
